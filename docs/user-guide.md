# ImageCreator ユーザーガイド

## はじめに

ImageCreator は、日本語のプロンプトから画像を生成できるオフライン動作の Windows 用 AI クライアントです。

## インストール

1. [GitHub Releases](../../releases) から `ImageCreator_X.Y.Z_x64-setup.exe` をダウンロード
2. ダブルクリックで実行
3. SmartScreen の警告が出たら「**詳細情報**」→「**実行**」をクリック（[詳細は README 参照](../README.md#-windows-smartscreen-の警告について)）
4. インストーラの案内に従って完了

## 初回セットアップ

アプリを初めて起動するとセットアップウィザードが表示されます。

| ステップ | 内容 |
|---|---|
| 1. ようこそ | アプリ概要の説明 |
| 2. ライセンス | Anima 非商用ライセンスへの同意 |
| 3. GPU 検出 | NVIDIA GPU の自動検出（VRAM 表示） |
| 4. データフォルダ | モデル・ランタイムの保存先選択（最低 12GB の空き必要） |
| 5. 確認 | 設定内容の確認 |
| 6. ダウンロード | 必要ファイル（約 10GB）の自動取得＋展開 |

ダウンロードは回線速度により 30 分〜2 時間かかります。途中で閉じても次回起動時に続きから再開可能です。

## 基本操作

### 画像を作る（シンプルモード）

1. メイン画面で **「ComfyUI 起動」** ボタンを押す（約 1〜3 分）
2. **「プロンプト変換 LLM 起動」** ボタンを押す（約 10〜20 秒）
3. プロンプト入力欄に**日本語で**作りたい画像のイメージを書く
   - 例: 「青空の下で笑う金髪の少女」
4. サイズ（正方形 / 縦長 / 横長）と枚数（1〜4 枚）を選ぶ
5. 「**画像を生成**」をクリック
6. 自動で日本語 → 英語タグ変換 → 画像生成が走る
7. 完了すると右側に画像が表示される

### 詳細モード

画面右上の「**シンプル / 詳細**」スイッチで詳細モードに切替できます。

詳細モードでは:
- 日本語プロンプトの変換結果（英語タグ）を手動編集できる
- ネガティブプロンプトを変更できる
- サンプラー、Steps、CFG などすべて手動制御
- バッチ生成（同じ設定で複数枚生成）が可能
- Turbo LoRA（高速生成モード）を ON/OFF できる
- 既定値リセットボタン

## ギャラリー

過去に生成した画像は **「ギャラリー」** タブで閲覧できます。

各画像をクリックすると詳細モーダルが開き、以下を確認できます:
- フルサイズ画像
- 入力プロンプト（日本語＋英語両方）
- 生成パラメータ（解像度、Steps、CFG、サンプラー、シード）
- 生成日時

主な操作:
- **★ ボタン**: お気に入り登録
- **「同じ設定で再生成」**: 全パラメータを生成画面に流し込む（詳細モードで開く）
- **🗑 ボタン**: 削除（ファイルも削除されます）

PNG ファイルにはメタデータ（tEXt チャンク）が埋め込まれているため、外部画像ビューアでもプロンプト等を確認できます。

## 設定

サイドバーの **「設定」** から以下を変更できます。

### 一般タブ
- **表示言語**: 日本語 / English（即時切替）
- **データフォルダ**: 保存先変更（要アプリ再起動）
- **初回セットアップ再実行**: ウィザードを再度開く

### モデルタブ
- 登録モデル一覧と各モデルの必須/任意ファイル取得状況
- ライセンス情報の確認

### 詳細タブ
- バージョン情報
- 設定ファイル・ログファイル等の保存パス
- **翻訳キャッシュをクリア**: プロンプト変換キャッシュ消去
- **設定をリセット**: 既定値に戻す（モデル・画像は残る）

## ログ閲覧

トラブル時は **「ログ」** タブから以下のログを参照できます:
- **アプリ**: ImageCreator 本体のログ
- **ComfyUI**: ComfyUI プロセスの出力
- **llama-server**: プロンプト変換 LLM の出力

各タブで:
- 自動更新（3 秒毎）
- 文字列フィルタ（部分一致）
- 全コピー（クリップボード）
- ログクリア

## アンインストール

Windows の「設定 → アプリ → インストールされているアプリ」から「ImageCreator」をアンインストールしてください。

アンインストーラに表示される **「アプリケーションデータを削除」チェックボックス**で、データの扱いを選びます:

- **チェック ON**: ユーザーデータ（設定・ギャラリー履歴・ログ）を削除し、
  それに連動してダウンロード済みモデル・ランタイム（約 10GB）も自動削除されます。
- **チェック OFF**: ユーザーデータもモデル・ランタイムも残します。
  再インストール時に設定・ギャラリーが引き継がれ、モデルの再ダウンロードも不要です。

いずれの場合もアプリ本体・ショートカット・レジストリ登録は削除されます。

## トラブルシューティング

### ComfyUI が起動しない / 応答しない

1. 設定→詳細タブで「アプリログ」のパスを確認
2. ログタブの「ComfyUI」を確認
3. nvidia-smi が動作するか確認（コマンドプロンプトで `nvidia-smi`）
4. NVIDIA ドライバが最新か確認

### プロンプト変換が失敗する

1. 「プロンプト変換 LLM」のステータスを確認
2. 起動していない場合は「起動」ボタンを押す
3. 起動完了に 10〜20 秒かかります
4. ログタブの「llama-server」で詳細確認

### VRAM 不足エラー

1. 解像度を下げる（1024² → 768²）
2. バッチサイズを 1 に
3. 他のアプリ（特に他の AI 画像生成ツール）を閉じる

### ダウンロード途中で失敗した

1. 「設定 → モデル」タブで不足ファイルを確認
2. 該当モデルの欠損ファイルを個別 DL（将来対応予定）
3. 当面はセットアップを再実行（既存ファイルは継続利用）
